GitHub Actions CI/CD Workflow Documentation

A step-by-step guide to deploying a static site with GitHub Actions.

Steps to Set Up the Workflow

1. Create a New GitHub Repository

Create a repository named gh-deployment-workflow and add basic files like index.html and README.md.

2. Set Up GitHub Actions Workflow

Create a .github/workflows/deploy.yml file to define the workflow for GitHub Pages deployment.

3. Configure Workflow Triggers

Set the workflow to trigger on changes to the index.html file on the main branch.

4. Add Deploy Step

Use the peaceiris/actions-gh-pages@v3 action to deploy the site to GitHub Pages.

5. Push Changes and Verify

Push your changes and verify that the site deploys successfully to GitHub Pages.

Created as a learning project for CI/CD with GitHub Actions
